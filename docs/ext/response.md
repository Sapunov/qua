## Описание ответа API

Если при выполнении запроса не было никаких ошибок, то API вернет ответ в следующем формате:

```
{
    "ok": 1,
    "response": {...} (или null)
}
```

Если при выполнении запроса возникла ошибка, ответ вернется в следующем виде:

```
{
    "ok": 0,
    "error": {
        "error_code": 100,
        "error_msg": "...",
    }
}
```
