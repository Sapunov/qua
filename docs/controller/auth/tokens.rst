Операции с токенами
===================

Получение нового токена
-----------------------

Получение нового JWT токена для выполнения авторизаванных
запросов к API

Формат запроса
~~~~~~~~~~~~~~

::

    POST /auth/tokens

Пример запроса
~~~~~~~~~~~~~~

::

    POST /auth/tokens HTTP/1.1

    {
        "username": "admin",
        "password": "qwerty"
    }


Параметры запроса
~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 10 5 20 5

   * - Название
     - Тип
     - Описание
     - Обязательный
   * - username
     - string
     - Логин пользователя, для которого нужно получить токен
     - Да
   * - password
     - string
     - Пароль пользователя
     - Да

Пример ответа
~~~~~~~~~~~~~

::

    {
        "ok": 1,
        "response": {
            "token": "<JWT токен>"
        }
    }

Параметры ответа
~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 10 5 20

   * - Название
     - Тип
     - Описание
   * - token
     - string
     - JWT токен

------------

Формат запроса

.. list-table::
